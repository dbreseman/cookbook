# Cookbook

* To view published recipes, see https://dbreseman.github.io/cookbook/.
* All recipes are [`.cook` files](https://cooklang.org/docs/spec/).
* For local preview, clone the repo and install the [Cooklang CLI](https://cooklang.org/docs/getting-started-commands/).
* The deployed site uses /.github/workflows/deploy.yml to update automatically upon commit.
